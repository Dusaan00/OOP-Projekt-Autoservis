# OOP Projekt - Autoservis

Aplikace autoservisu, která běží v konzoli. Aplikace byla vytvořena v C++ na základě principů OOP jako školní projekt zaměřený na OOP. Projekt eviduje zákazníka, vozidla, mechaniky, servisní úkony a celý autoservis s objednávkami, s využitím těch nejdůležitějších principů OOP jako jsou konstruktory, destruktory, dědičnost, abstraktní třídy, polymorfismus, přetížené metody a virtuální funkce.
